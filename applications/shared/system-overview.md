# ORGANVM — System Overview

## What It Is

ORGANVM is an eight-organ creative-institutional system coordinating 149 registry entries spanning the organ/meta GitHub organizations plus personal/local entries. Each organ governs a domain — from epistemological theory (ORGAN-I) through generative art (ORGAN-II), commercial products (ORGAN-III), orchestration governance (ORGAN-IV), public documentation (ORGAN-V), community infrastructure (ORGAN-VI), marketing distribution (ORGAN-VII), to meta-system governance (META).
